Amazon Web Services
CI/CD Project

This project is a demonstration of CI/CD concepts by creating a static web hosting S3 bucket with CloudFormation and using CloudBuild as a form of automated testing. This is all orchestrated by CodePipeline

Services

  • CodePipeline
  • CodeBuild
  • CloudFormation

Prerequisites

  1. The website.json template should be used in CloudFormation to create a S3 bucket which has two unique properties. First, the access control property is set to public read and second, website hosting is enabled.
  2. Create a file named index.html and upload it to the GitHub repository branch
